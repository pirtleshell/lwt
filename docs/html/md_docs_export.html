About LWT Export Templates for "Flexible Exports"

An export template consists of a string of characters. Some parts of this string are placeholders (beginning with **"%", "$" or "\\"**) that are <strong>replaced</strong> by the actual term data, <strong>see the following table</strong>. For each term (word or expression), that has been selected for export, the placeholders of the export template will be replaced by the term data and the string will be written to the export file. <strong>A template must end with</strong> either **"\n"** (UNIX, Mac) or **"\r\n"** (Windows). <strong>If you omit this, the whole export will be one single line!</strong> If the export template is <strong>empty, no terms of this language</strong> will be exported. <table class="markdownTable"> <tr class="markdownTableHead"> <th class="markdownTableHeadNone"> Placeholders
